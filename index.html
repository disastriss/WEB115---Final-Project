What I learned in HTML

The biggest takeaway I have from my time working with HTML is that it is essentially the skeleton of the project. It is what holds together all the meat of the project. Without it, there would not be structure or order. There would be no content to manipulate with CSS or Javascript. Not only that but being able to link to other pages is what makes the internet the internet. HTML is the reason you can surf the web. Though it was the easiest coding language for me to learn, it does feel like one of the most important to know.

What I learned in CSS

CSS for me has been a bit more confusing than HTML, but much more exciting to use. In high school, I loved adding flair to my basic word documents or using photoshop to create designs to incorporate into my assignments, so as difficult as CSS has been, I’ve been loving learning how to use code to make websites beautiful. The most useful CSS method I learned was being able to organize elements onto the page using the ‘flex’ property. The biggest challenge in creating a website is how to get everything you need onto the page and make it aesthetically pleasing. There are so many different layouts that can be done to look good, and most are not possible without the ‘flex’ property.

How I incorporated JavaScript

When you load my webpage, if you look closely you can see little pink circles floating up from the bottom. This was done with JavaScript and CSS! With the help of Google and what I learned through our assignments in Mindtap, I found two functions I was able to incorporate into my index.html with a script tag. The first function was for the creation of the “stars” where I used a ‘for’ loop and animation. The second function was a randomNumber function, so the little pink “stars” would appear at random places and move at different random speeds. Then I added some CSS to give them shape and color. I still have a long way to go with fully understanding JavaScript, since I relied heavily on code that was already created, but it was really exciting to understand enough to get it to work on my page exactly as I imagined it would.
